# FreshCart - Responsive Grocery & Price Catalog Website

A clean, modern, and lightning-fast front-end landing page designed for local grocery stores, supermarkets, or fresh produce businesses. This project showcases a fully responsive layout that works flawlessly across desktops, tablets, and smartphones.

## 🚀 Live Demo
See the live website in action here: 
👉 **[https://bala-business.github.io/grocery/index.html](https://bala-business.github.io/grocery/index.html)**

---

## ✨ Features
* **100% Responsive Design:** Beautifully optimized for all device viewports (Desktop, Tablet, and Mobile).
* **Smart Floating Header:** Seamless, smooth-scrolling navigation bar that dynamically adjusts to provide a compact experience on mobile screens.
* **Organized Price Catalog:** A clean, easy-to-read table layout categorizing grocery items with distinct typography for quick browsing.
* **Modern CSS Architecture:** Structured with custom CSS variables, clean reset blocks, responsive flex/grid layouts, and optimized media queries.
* **Performance Focused:** Built using pure HTML5 and vanilla CSS3—zero bloated frameworks, ensuring instant loading speeds.

---

## 🛠️ Tech Stack Used
* **HTML5:** Semantic markup structure.
* **CSS3:** Custom layouts, scrolling transitions, typography, and responsive media queries.
